# Hello

Welcome to a blog scaffolded with `pyssg new --theme blog`.

This is the home page (`content/index.md`). Your posts live under
`content/blog/` and are listed, newest first, on the [Blog](/blog/) page. Add a
new post with:

```bash
pyssg new post "My first post"
```

Then run `pyssg serve` to preview with live reload.
