This is a documentation site scaffolded with `pyssg new --theme docs`.

The sidebar on the left is built automatically from the folder structure under
`content/`. Drop a Markdown file anywhere and it becomes a page; add an
`index.md` to a folder to give that section a landing page.

## Next steps

- Edit this file at `content/index.md`.
- Read the starter [Guide](/guide/) and replace it with your own.
- Run `pyssg serve` to preview with live reload.
