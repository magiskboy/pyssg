# Guide

Practical, hands-on documentation for getting a site up and running with pyssg.

- [Getting started](/guide/getting-started/) - install, scaffold and build your
  first site.
- [Configuration](/guide/configuration/) - the `pyssg.config.py` file and the
  `Config` object.
- [Dev server](/guide/dev-server/) - `pyssg serve` for live preview with
  rebuild-on-change.
- [Deploy](/guide/deploy/) - recipes for GitHub Pages, Netlify and Cloudflare
  Pages.

If you want the conceptual picture first, jump to the
[Architecture](/architecture/) section.
